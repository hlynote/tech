# 欢迎来到 Tech Note

这是项目里的第一篇技术博客示例文章。

## 你现在可以做什么

- 在 `content/posts` 下新增 `.md` 或 `.mdx` 文件
- 通过 frontmatter 配置标题、时间、摘要、标签
- 在博客列表页和详情页自动渲染内容
